Academia CSS

A simple CSS styling for the web that is heavily inspired by Academia's obsession with A4 paper size and the standard Times New Roman, 12px, 1.5 spacing.

Kitchen Sink

There are several heading types: from h1 to h6. All headings have a 2em top margin.

h1 is 6em

h2 is 4em

h3 is 2em

h4 is 1.5em

h5 is 1.2em
h6 is 1em

paragraph text is 1em
